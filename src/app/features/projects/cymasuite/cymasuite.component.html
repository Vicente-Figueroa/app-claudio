Cymasuite

Mi experiencia como científico de datos en Cymasuite, me permitió desarrollar soluciones predictivas clave. Diseñé un predictor de cuentas bancarias para anticipar movimientos financieros y un sistema para detectar facturas fraudulentas, mejorando la seguridad y confiabilidad de las transacciones de los usuarios. Además, implementé sistemas inteligentes con métricas de mantenimiento para optimizar la eficiencia y la toma de decisiones estratégicas.
